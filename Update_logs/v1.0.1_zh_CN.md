# Tuanzi's Server Mod 更新日志 - v1.0.1

**版本号:** 1.0.1  
**发布日期:** 2026-04-26  
**适配版本:** Minecraft 1.21.1 (Fabric)

---

## 🚀 版本摘要
本次更新引入了全新的**称号系统 (Title System)**，并增加了多项统计指标。玩家现在可以通过获得和佩戴独特的称号来彰显个性。

---

## 🛠 新增内容

### 🏷️ 称号系统 (Title System)
- **称号佩戴与展示**:
    - 玩家可以自由佩戴已拥有的称号，称号将显示在玩家名称前。
    - 称号在**聊天栏、Tab 列表、玩家头顶**及系统消息中均可见。
    - 支持彩色称号名称，且经过特殊设计，确保称号颜色不会溢出导致用户名颜色改变。
- **获得称号**:
    - **特制命名牌**: 玩家可以通过右键空气或方块使用“特制命名牌”来获得新的称号（使用后消耗）。
    - **指令发放**: 管理员可直接向玩家发放称号。
- **玩家指令 (`/titles`)**:
    - `/titles list`: 查看已拥有的称号并快速切换。
    - `/titles set <id>`: 切换至指定称号。
    - `/titles clear`: 卸下当前称号。
- **管理员指令 (`/titleadmin`)**:
    - **称号创建**: 创建新称号并自动获得对应的命名牌。
    - **管理与分发**: 查看所有称号、获取命名牌，或一键给单人/全服玩家（含离线玩家）发放称号。
    - **离线通知**: 离线玩家在获得称号后，将在下次上线时收到获得通知。
    - **安全删除**: 管理员删除称号时，会自动清除全服玩家对该称号的拥有记录，防止死灰复燃。

### 📊 统计系统增强 (Statistics Update)
- 增加了 4 项全新的称号相关统计数据：
    - **称号持有数量**: 玩家当前拥有的称号总数。
    - **切换称号次数**: 玩家使用指令更换称号的频率。
    - **通过管理员获得的称号数**: 由管理员直接发放获得的称号数量。
    - **通过命名牌获得的称号数**: 通过使用消耗性命名牌激活的称号数量。

---

## 💎 技术特性
- **纯原版机制**: 基于 Minecraft 计分板队伍 (Scoreboard Teams) 实现，**零 Mixin** 侵入，极高兼容性。
- **离线排队通知**: 引入离线消息队列，确保离线期间的荣誉发放不被遗漏。
- **高性能**: 数据结构优化，大规模称号删除与统计更新对服务器性能影响微乎其微。
