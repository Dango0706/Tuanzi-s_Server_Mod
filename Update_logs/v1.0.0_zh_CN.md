# Tuanzi's Server Mod 更新日志 - v1.0.0

**版本号:** 1.0.0  
**发布日期:** 2026-04-25  
**适配版本:** Minecraft 1.21.1 (Fabric)

---

## 🚀 版本摘要
这是 Tuanzi's Server Mod 的第一个正式版本。本模组旨在为 Minecraft 服务器提供全方位的增强功能，涵盖了身份验证、经济体系、智能商店及全维度数据统计四大核心模块，且完全无需客户端安装任何模组。

---

## 🛠 详细更新内容

### 🔐 身份验证 (Auth)
- **混合验证模式**: 引入智能识别系统，正版玩家免密登录，盗版玩家通过注册登录流程访问。
- **增强白名单**: 基于 UUID 映射的增强型白名单，支持离线玩家 UUID 生成与管理。
- **安全沙箱**: 玩家在登录前将被限制移动、交互及伤害，确保服务器地理信息及物理安全。

### 💰 经济系统 (Economy)
- **多钱包架构**: 支持多种货币并行（如金币、点券），管理员可自由定义货币属性。
- **动态国际化**: 货币名称及交易反馈会根据玩家客户端设置的语言（中文/英文）自动切换。
- **离线转账**: 支持向离线但已在服务器注册的玩家进行转账操作。

### 🏪 智能商店 (Shop)
- **引导式创建**: 创新的木牌交互系统，通过聊天框按钮和输入引导玩家轻松完成商店设置。
- **动态定价算法**: 
    - 引入 **BUY (收购)** 和 **SELL (出售)** 两种动态价格模式。
    - 价格随市场库存（S）与平衡系数（K）动态波动，实现自动化经济调节。
- **视觉展示**: 采用非持久化 `ItemEntity` 实现物理悬浮物品展示，具备防吸取、零存档负担等特性。

### 📊 统计与排行榜 (Statistics)
- **全维度追踪**: 精确记录包括挖掘量、击杀数、在线时间、死亡距离等在内的 20+ 项玩家数据。
- **动态排行榜**:
    - **悬浮文字 (Floating Text)**: 可在主城等区域创建实时更新的 3D 文字排行榜。
    - **计分板轮换**: 侧边栏计分板支持自动轮换展示不同维度的统计数据。

---

## 💎 核心技术特性
- **纯服务端设计**: 玩家无需安装任何模组即可享受所有功能。
- **i18n 全面支持**: 所有的系统提示、指令反馈、UI 文本均适配中英双语。
- **高性能架构**: 统计数据异步持久化，商店实体节流维护，确保服务器 TPS 稳定。
