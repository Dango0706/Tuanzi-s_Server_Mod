# Tuanzi's Server Mod Update Log - v1.0.0

**Version:** 1.0.0  
**Release Date:** April 25, 2026  
**Minecraft Version:** 1.21.1 (Fabric)

---

## 🚀 Version Summary
This marks the first official release of Tuanzi's Server Mod. Designed as a comprehensive server-side enhancement, this mod integrates four core modules: Authentication, Economy, Smart Shop, and Total Statistics, all without requiring any client-side installations.

---

## 🛠 Detailed Updates

### 🔐 Authentication (Auth)
- **Mixed Mode Verification**: Automatically distinguishes between Mojang premium accounts (password-less login) and non-premium accounts (registration/login required).
- **Enhanced Whitelist**: A UUID-mapped whitelist system with support for offline player UUID generation.
- **Security Sandbox**: Restricted movement, interaction, and damage for players prior to successful login to ensure server safety.

### 💰 Economy System
- **Multi-Wallet Architecture**: Support for multiple currencies (e.g., Gold, Credits) with admin-defined properties.
- **Dynamic i18n**: Currency names and transaction feedback automatically adapt to the player's client language (English/Chinese).
- **Offline Transactions**: Ability to transfer funds to players who are currently offline but registered.

### 🏪 Smart Shop
- **Guided Creation**: An interactive sign-based system that uses chat buttons and input prompts to guide users through shop setup.
- **Dynamic Pricing Algorithm**: 
    - Features **BUY** and **SELL** modes.
    - Prices fluctuate dynamically based on stock levels (S) and a balance coefficient (K) to automate market regulation.
- **Visual Display**: Non-persistent `ItemEntity` displays for floating items, featuring anti-pickup logic and zero storage impact.

### 📊 Statistics & Leaderboards
- **Total Tracking**: Precision tracking for 20+ metrics including blocks mined, kills, playtime, and death distance.
- **Dynamic Displays**:
    - **Floating Text**: Create real-time updating 3D text leaderboards in-game.
    - **Scoreboard Rotation**: Sidebar scoreboards that automatically rotate through different statistical categories.

---

## 💎 Core Technical Features
- **Server-Side Only**: Enjoy all features without any client-side mod requirements.
- **Full i18n Support**: System prompts, command feedback, and UI text fully localized for English and Chinese.
- **High Performance**: Asynchronous persistence for statistics and throttled maintenance for shop entities to ensure stable Server TPS.
