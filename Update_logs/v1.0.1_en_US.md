# Tuanzi's Server Mod Update Log - v1.0.1

**Version:** 1.0.1  
**Release Date:** April 26, 2026  
**Minecraft Version:** 1.21.1 (Fabric)

---

## 🚀 Version Summary
This update introduces the brand-new **Title System** and adds several new statistical metrics. Players can now showcase their personality by obtaining and wearing unique titles.

---

## 🛠 New Features

### 🏷️ Title System
- **Wearing & Display**:
    - Players can freely equip owned titles, which appear before their name.
    - Titles are visible in **Chat, Tab List, above player heads**, and in system messages.
    - Supports colored title names with special design to prevent color bleed into usernames.
- **Obtaining Titles**:
    - **Custom Name Tags**: Players can obtain new titles by right-clicking air or blocks with a "Title Name Tag" (consumed on use).
    - **Admin Commands**: Direct title granting by administrators.
- **Player Commands (`/titles`)**:
    - `/titles list`: View owned titles and switch quickly.
    - `/titles set <id>`: Switch to a specific title.
    - `/titles clear`: Unequip current title.
- **Admin Commands (`/titleadmin`)**:
    - **Title Creation**: Create new titles and automatically receive the corresponding name tag.
    - **Management & Distribution**: View all titles, get name tags, or grant titles to individual or all players (including offline ones).
    - **Offline Notifications**: Offline players will receive a notification upon their next login if they were granted a title while away.
    - **Secure Deletion**: Deleting a title automatically removes ownership records from all players to prevent unauthorized use.

### 📊 Statistics Enhancement
- Added 4 new title-related statistical metrics:
    - **Titles Owned**: Total number of titles currently owned by the player.
    - **Title Switches**: Frequency of title changes using commands.
    - **Titles from Admin**: Number of titles granted directly by administrators.
    - **Titles from Items**: Number of titles activated via consumable name tags.

---

## 💎 Technical Features
- **Vanilla-Based**: Implemented via Minecraft Scoreboard Teams with **Zero Mixin** intrusion for maximum compatibility.
- **Offline Queued Notifications**: Introduced an offline message queue to ensure players don't miss their rewards.
- **High Performance**: Optimized data structures ensure that large-scale title deletions and statistical updates have minimal impact on server performance.
